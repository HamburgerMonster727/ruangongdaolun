# 自制小游戏：大战食人族

玩家将通过用枪进行射击，点击鼠标发射子弹，上下左右控制移动方向，碰到食人族会被抓，按空格重新进行游戏，当击杀一定数目的小食人族后，会刷新一个大食人族，把所有食人族击杀后，游戏获胜
